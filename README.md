# Eunjeong's protfilio
> ['핀터레스트'](https://www.pinterest.co.kr/pin/421649583869385322/)에서 포트폴리오의 디자인을 얻어 만들었습니다.

## 목적
저의 이력 및 가치를 알리기 위해 제작했습니다.

## 실행 및  설명
이 프로젝트는 **Create React App**으로 개발했습니다.
1. 프로젝트를 clone 받으세요.
2. **npm start**로 프로젝트를 실행시켜 확인해주세요.
